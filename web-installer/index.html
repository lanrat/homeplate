Instructions

  1. Connect your Inkplate to your computer via USB.
  2. Select your board variant above.
  3. Click Install HomePlate and select the serial port.
  4. After flashing, the device will create a HomePlate-Setup WiFi network — connect to it to configure your settings. See the Setup Guide for details.

Notes

  • Requires Chrome or Edge on desktop (Web Serial API).
  • You may need to install CP2102 USB drivers for your operating system.
  • First-time installs will erase all settings. Updating existing installations will preserve your configuration.
  • After installation, use the Logs & Console option in the install dialog to view serial output for debugging.
